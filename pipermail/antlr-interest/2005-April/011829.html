I need to parse output of a custom made program. It can produce
messages consisting of arbitrary ascii chars. Thus, I tried to make my
lexer rule:

protected LETTER : (('!'..'~')) => ('a'..'z'|'A'..'Z')
                                   | (':')   { $setType(COLON); }
                                   | ('\\')  { $setType(BSLASH); }
                                   | ('(')   { $setType(LPAREN); }
                                   | (')')   { $setType(RPAREN); }
                                   | ('!'..'~') { $setType(ASCIISYM); }
;
