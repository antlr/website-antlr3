FWIW I found a simple solution to this problem...

Rather than use += I'm using code blocks in the group itself.


i.e. Instead of this
   call
   : ^(CALL IDENTIFIER (e+=INT)*)
   {
      $e ...
   };

I'm now doing this
   call
   : ^(
         CALL
         IDENTIFIER
         (
            e { list.Add( e ); }
         )*
      )

Its not as easy to read but it works.
