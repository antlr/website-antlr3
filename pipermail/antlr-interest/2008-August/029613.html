Hi Fred,

You should be able to use predicates, something like this:

COMMAND
      : 'create' | 'delete'                     { inCommand = true; }
      ;

COMMAND_TOKEN
      : {inCommand}?=>
          (options{greedy=false;} : . ) NEW_LINE  { inCommand = false; }
         ;

Matt.
